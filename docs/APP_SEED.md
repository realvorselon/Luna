# App Seed Note

Luna may later become a calm companion for small projects. This is only a tiny direction marker for later, not a roadmap, build plan, feature list, or promise.

## Smallest Honest Shape

If Luna ever grows into an app, the smallest honest shape could be:

- Hold **one small project at a time** so the person is not asked to manage everything at once.
- Begin with a **gentle setup** that asks what the project is and what matters about it.
- Keep a simple **saved project state** so returning does not mean starting over.
- Name a current mode, such as **orient**, **grow**, **align**, **prune**, **reflect**, or **rest**.
- Offer **one suggested next action** that is small, kind, and possible now.
- Show a short history of **what changed last**.
- Include a reminder of **what not to worry about yet**.

## What This Is Not

This note does not start the app. It does not add code, styling, a dashboard section, accounts, pricing, a backend, a database, an LLM plan, technical architecture, or a roadmap expansion.

For now, Luna should rest or review this note before adding any app feature, dashboard feature, visual styling, backend, account system, or roadmap expansion.
