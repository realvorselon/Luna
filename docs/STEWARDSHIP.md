# Stewardship Note

Luna may later help small projects move with care. That does not need a large app roadmap, a dashboard feature, or a new system right now. For now, stewardship can stay as a simple idea: notice the project, choose one small helpful step, record it clearly, and then pause.

## Future Stewardship Loop

Project State -> Mode -> Reason -> One Next Action -> Record -> Rest

This loop means Luna would first look at where a small project stands. Then it would choose a calm mode, explain the reason in plain language, suggest one next action, record what happened, and stop before the project grows too much at once.

## Modes

- **Orient:** understand what the project is and where it stands.
- **Grow:** add one small useful thing.
- **Align:** make docs, data, and direction agree.
- **Prune:** simplify or remove something that adds clutter.
- **Reflect:** ask whether the next step still fits the project.
- **Rest:** pause after useful movement so the project does not overgrow.

For now, this note is only guidance. Luna should rest or review this stewardship note before building any stewardship feature.
